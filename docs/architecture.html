Presentation Layer

02
Orchestration Layer
▼

NeuroCrew Orchestrator

Manages multi-agent conversations via AutoGen RoundRobinGroupChat. MaxMessageTermination(12) + TextMentionTermination("TERMINATE").

src/orchestrator.py

SingleAgentChat

Direct consultations with individual agents. Faster for single-purpose tasks. 3-message termination limit.

src/orchestrator.py

Model Client Factory

Creates OpenAIChatCompletionClient based on config. Supports OpenAI API and local Ollama/LM Studio via OpenAI-compatible endpoint.

src/orchestrator.py:31

AI Agent Layer
▼

BaseNeurologistAgent

Abstract base class defining shared interface: system_message, tools, agent_type. All 6 agents extend this contract.

src/agents/base_agent.py

Agent Prompts

Each agent has a detailed system prompt with CoT reasoning, few-shot examples, structured output templates, self-review checklists, and guardrails.

src/agents/*.py

Helper Methods

Clinically accurate reference methods: get_red_flags(), calculate_trend(), validate_assessment_score(), get_first_line_treatments(), etc.

src/agents/*.py
04
Data & Validation Layer
▼

Pydantic v2 Models

Type-safe schemas: Patient, Visit, VitalSigns, NeurologicalAssessment, Medication, PrognosisAnalysis with strict field validation.

src/models/schemas.py

Enum Definitions

Gender, NeurologicalCondition (7 types), PrognosisTrend, SeverityLevel. Ensures categorical data consistency across the system.

src/models/schemas.py

Test Patient Data

05
Infrastructure Layer
▼

HIPAA Audit Logger

PHI hashing via SHA256, separate log streams (audit, PHI access, agents), 365-day audit / 7-year PHI retention, JSON structured format.

src/logging/audit_logger.py

LLM Telemetry

Token counting, cost estimation per model, session metrics, per-agent breakdowns, JSONL event log, human-readable cost reports.

src/logging/telemetry.py

Configuration

Environment-based config via .env files. Supports LLM provider switching, model selection, log levels, and database URLs.

src/config/settings.py

Prompt Engineering Techniques

CoT

Chain-of-Thought

Numbered step-by-step reasoning process unique to each agent's specialty

FS

Few-Shot Examples

Complete input/output examples with realistic clinical data in every prompt

SO

Structured Output

Named templates: CLINICAL ASSESSMENT, PROGNOSIS ANALYSIS, VALIDATION REPORT

SR

Self-Review

Verification checklists agents must complete before submitting responses

CC

Confidence Calibration

Required confidence levels -- High/Moderate/Low or 0.0-1.0 numeric scores

RG

Reference Grounding

Score ranges, clinical benchmarks, dosage tables embedded in prompts

GR

Guardrails

Critical Rules sections with explicit safety boundaries and prohibitions
