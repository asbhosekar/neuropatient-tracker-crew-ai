Orchestrator: NeuroCrew

The NeuroCrew orchestrator coordinates all 6 agents through a RoundRobinGroupChat, managing conversation flow, streaming output, logging, and termination.

Agent Creation

Instantiates all 6 specialized agents with a shared LLM model client (OpenAI or local). Each agent is an AssistantAgent with custom system prompt and tools.

Team Coordination

Uses RoundRobinGroupChat so agents take turns responding, building on each other's analysis for collaborative clinical reasoning.

Termination Conditions

MaxMessageTermination(12) stops after 12 messages. TextMentionTermination("TERMINATE") allows early exit when agents reach consensus.

Streaming & Logging

Real-time message streaming to UI. Every agent message logged with correlation IDs, PHI access tracked, and telemetry recorded per call.

Conversation Flow (Prognosis Analysis)

1
User submits patient data via Streamlit UI or CLI menu
2
Orchestrator prepares task prompt, initializes RoundRobinGroupChat with all agents
3
Neurologist provides initial clinical assessment and differential diagnosis
4
Prognosis Analyst analyzes longitudinal data, calculates trends and trajectories
5
Treatment Advisor recommends medications, flags interactions, suggests adjustments
6
QA Validator validates all data, checks clinical logic and score consistency
7
Report Generator compiles findings into a structured clinical report
8
Clinical Architect reviews data model compliance, HIPAA adherence
9
Round-robin continues until MaxMessage(12) or TERMINATE signal
